Avec un petit groupe de 5 étudiants, notre tâche était de créer un site de e-commerce pour l'entreprise Créa-Cuisine. Ce projet est le premier projet réaliser pour un professionnel dont je prends part. Le travail se doit donc d’être sérieux et de respecter les délais fixés. Nous avons utilisé le CMS wordpress.

Le projet Laravel avait comme but de nous faire travailler avec la méthode de travail SCRUM sur le développement d'un site d'e-commerce fictif. Notre équipe de 5 personnes devait s'organiser du mieux possible pour présenter à un client fictif, toutes les semaines, l'avancement du projet.

Le but de ce jeu est de trouver ce qu'il y a de dissimulé derrière les tuiles .Pour ce faire il faut en détruire le moins possible.

DashRaider est principalement un jeu de puzzle/action. Dans DashRaider vous jouez le raider, votre but est de traverser le donjon, en complétant chaque pièce par des dashes. Sachez cependant que votre nombre de dashes est limité, tout comme votre santé. Le donjon a trois étages avec cinq pièces chacune que vous pouvez explorer et compléter.

Développé sans CMS, uniquement en html, css et javacript.
